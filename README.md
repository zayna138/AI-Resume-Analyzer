# AI Resume Analyzer

## Project Overview
This project is an AI-based Resume Analyzer developed using Python and Streamlit.

## Features
- Upload resume PDF
- Detect technical skills
- Resume scoring system
- Resume suggestions
- ATS-style analysis

## Technologies Used
- Python
- Streamlit
- PyPDF2

## How to Run

1. Install libraries:
pip install streamlit PyPDF2

2. Run project:
streamlit run app.py
